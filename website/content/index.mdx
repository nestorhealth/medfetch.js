# Introduction

Welcome to the medfetch.js docs!

## What is Medfetch.js?
Medfetch.js is an opinionated javascript [FHIR](https://hl7.org/fhir/) data fetcher built on top of
modern [Web APIs](https://developer.mozilla.org/en-US/docs/Web/API).
